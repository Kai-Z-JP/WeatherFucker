# WeatherFucker

## これは何？
雨を絶対に降らなくするBukkit Pluginです。CraftBukkit/Spigot APIを使用していないのでかなり広いバージョンで使えます。
1.11以降では/gamerule doWeatherCycle [true|false]があるのであまり必要性は無いと思います。

## 動作確認済み環境
- Mohist(1.7.10) 45
- Dionysus(1.12.2) v0.1.15
- Paper(1.17.1) 140

## コマンド
- /wf [enable|disable]
WeatherFuckerを有効/無効にします。デフォルトは有効です。
